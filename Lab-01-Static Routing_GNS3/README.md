# Lab 01 – Rete LAN con IP statici

## Obiettivo
Creare una rete LAN con due host e uno switch e verificare la connettività.

## Topologia
PC1 —— Switch —— PC2

## Configurazione IP
- PC1: 192.168.10.10 /24
- PC2: 192.168.10.20 /24

## Procedura di test
- Verifica collegamenti fisici
- Ping ICMP riuscito
- Analisi ARP e ICMP in Simulation Mode

## Conclusioni
Rete LAN funzionante e correttamente testata.
